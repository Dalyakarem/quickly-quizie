# Quickly Quizie

Quickly Quizie is a React trivia game where users can choose a category and difficulty, answer timed multiple-choice questions, and see their final score at the end. The app uses the Open Trivia DB API, includes a 15-second countdown timer, randomizes answer order, and saves the top scores using localStorage.

## Features
- Dynamic trivia questions from Open Trivia DB
- Category and difficulty selection
- 15-second timer for each question
- Automatic transition when time runs out
- Shuffle logic for answer choices
- Results screen with score feedback
- High-score leaderboard with localStorage
- Loading and error handling
- Responsive layout

## Technologies Used
- React
- JavaScript
- CSS
- Open Trivia DB API
- localStorage

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
